My electronics and embedded systems projects

Sensor network

School project

In this project I had to learn how the CAN, I2C and SPI buses worked since all three protocols were used in the different sensors. I had to manage the transmission of information from 5 sensors (brightness, proximity, wind speed, humidity and gyroscope) simultaneously via a single CAN bus and display the data in real time on an interface.

Motor control

School project

During my 4th year, we had to realize the servo-control of a synchronous motor with a vector control. During this project I learned how to build PID controller and create command interfaces using Qt. The informations were transmitted from the computer to the STM32-Nucleo microcontroller via Zigbee.

Robot

School project

In this project we had to code the robot. We were using a real-time operating system. The goal of the project was to distribute the different tasks and to manage the priorities in the interruptions created by the different sensors. We also had to create a control interface and a tracking of a person via image/color recognition by the camera.
